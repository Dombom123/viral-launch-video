# Backend - Viral Video Generator

This is the backend for the Viral Video Generator, built with Python and Google's Agent Development Kit (ADK).

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## ADK Integration

We are using the Google Agent Development Kit for the multi-agent system.
Reference: https://google.github.io/adk-docs/tutorials/agent-team/

## Agents Structure

- **Research Agent**: Compiles info on product, market, audience.
- **Script Agent**: Generates viral video scripts.
- **Storyboard Agent**: Generates visual references.
- **Video Agent**: Handles generation and editing.
